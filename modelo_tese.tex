\section{Section of bibliographic revision}
\label{section:revbib}

Some ways to cite:

According to... bla bla bla \cite{Doe2020}. With page: \cite[p.~1]{Doe2020}. With page range: \cite[p.~1-2]{Doe2020}.

Many authors in the same citation. Bla Bla Bla. \cite{Doe2020,Brown2023report,Doe2021masters,ExampleOrg2023,Jones2019,JonesUnpublished2020,Miller2024phd,Smith2022proc,SmithTeam2021}.

If need to cite only the year. \citeyear{Doe2020}.

If need to city the title and the date of some text. \citetitle{Doe2020}.

Inline cite is useful when referencing an author as a subject in the middle of a sentence. \textcite{Doe2020} says something really interesting.

Full bibliographic reference (useful in reports, for instance)

\noindent\fullcite{Doe2020}

\longquote{
This is an example of a quote. It can span multiple lines and is formatted to stand out from the main text. \lipsum[20]. bla blabla. \cite[p.~20]{Doe2023}
